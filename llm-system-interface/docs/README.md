Gemini 2.5 Flash-Lite (Free Tier)

Max speed: 15 requests per minute (RPM)
Max usage: 1,000 requests per day
Token limit: 250,000 tokens per minute (shared between input and output)
Context window: Up to 1 million tokens per request (very large context)
No credit card required
Fastest of the Gemini models
Good for most school projects and high-usage needs
============================================

Limitations:

If you send more than 15 requests in a single minute, you’ll be blocked until the next minute.
If you send more than 1,000 requests in a day, you must wait until the next day for your quota to reset.
All API keys in the same Google Cloud project share the same quota—creating extra keys does NOT give you more free requests.
Each request (API call) counts toward your limits, no matter how many questions you ask in one prompt.
You cannot exceed 250,000 tokens per minute (input + output combined).
============================================
Summary:

1 million tokens = max size for a single request
250,000 tokens per minute = max total usage rate across all requests
