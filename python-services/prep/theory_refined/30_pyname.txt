  Reads sys.argv (the actual command-line words the user typed) and returns a namespace object. After
  this, args.clean is either True or False depending on whether --clean was passed.

  ---
  asyncio.run(main(clean=args.clean))

  main is an async function (defined elsewhere with async def main(...)). Async functions don't run
  immediately when called — they return a coroutine object. asyncio.run() starts the event loop, runs
  that coroutine to completion, then exits the loop. It's the standard entry point for async Python
  programs.

  clean=args.clean passes the boolean as a keyword argument into main.
