
  MCP = Model Context Protocol. It's a standard plug — like USB-C but
  for giving AI models access to external data and tools. Defined by
  Anthropic in 2024, now open and adopted broadly.

    4. Do they ALWAYS work with an LLM?

  No — MCP requires an MCP client, which is usually an LLM host.

  Without LLM:  MCP server is just a running process doing nothing
  With LLM:     LLM client discovers the server → calls tools → gets
  results → reasons about them

  But the server itself has no dependency on an LLM. You can call an MCP
   server directly (e.g., via CLI or code) to test it. The LLM is on the
   client side, deciding when and how to call the server's tools.

  LLMs that support MCP today: Claude (Anthropic), Cursor, Cline,
  Continue, Zed, Windsurf, and others via SDKs.
